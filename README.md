# Salary Prediction using Machine Learning

## 📌 Project Overview

This project predicts whether an individual's income is **greater than $50K or less than or equal to $50K** based on demographic and employment-related features.
The model is trained using the **Adult Income Dataset** and applies machine learning techniques to analyze patterns and make predictions.

The goal of this project is to demonstrate the use of **data preprocessing, feature engineering, and classification algorithms** to build a predictive model.

## ⚙️ Workflow

1. **Data Collection**
   Uses the Adult Income dataset containing demographic attributes.

2. **Data Preprocessing**

   * Handling missing values
   * Encoding categorical variables
   * Feature scaling

3. **Model Training**
   Machine learning models are trained to classify salary categories.

4. **Prediction**
   The trained model predicts whether income is:

   * `>50K`
   * `<=50K`

## 📊 Example Attributes Used

* Age
* Education Level
* Occupation
* Workclass
* Hours per week
* Marital status

These features help the model learn patterns related to income levels.
